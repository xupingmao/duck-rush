# json提取工具

功能：从json字符串中提取指定的字段值


# 布局

左右布局
- 左侧：
  - textarea输入框：用于输入json字符串，宽度100%
  - input输入框：用于输入要提取的字段路径
  - 按钮组：
    - 提取按钮：点击后触发提取操作
    - 清空按钮：点击后清空输入框和输出框
    - 格式化按钮：点击后格式化json字符串
- 右侧：
  - textarea输出结果框：用于显示提取结果
  - 按钮组：
    - 复制按钮：点击后复制输出结果到剪贴板
- footer：
  - 显示工具版本号和更新日期
  - 显示工具使用说明和注意事项

# 功能实现

- 功能实现文件 `./json-extract.html`
- 输入json字符串后，点击提取按钮，即可提取指定字段的值
- 提取结果实时显示在右侧输出框中，通过toast消息提示
- 支持嵌套字段提取，如：`data.user.name`
- 支持数组字段提取，如：`data.users[0].name`
- 支持通配符提取，如：`data.*.name`
- 支持正则表达式提取，如：`data.*.name:/^[A-Z]/`

约束条件
- 不依赖任何外部库
- 使用ES5语法实现，保证兼容性

# 示例

输入json字符串：
```json
{
  "data": {
    "user": {
      "name": "张三",
      "age": 30
    },
    "users": [
      {
        "name": "李四",
        "age": 25
      },
      {
        "name": "王五",
        "age": 35
      }
    ]
  }
}
```

输入字段名：`data.user.name`
输出结果：
```json
张三
李四
王五
```
