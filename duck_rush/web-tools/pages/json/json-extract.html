使用说明

  • 在左侧输入框中粘贴JSON字符串
  • 在字段路径输入框中输入要提取的字段路径
  • 点击"提取"按钮获取结果
  • 支持嵌套字段，如：data.user.name
  • 支持数组索引，如：data.users[0].name
  • 支持通配符*，如：data.*.name
  • 支持正则表达式，如：data.*.name:/^[A-Z]/
  • 点击"格式化"按钮可以美化JSON格式
  • 点击"复制"按钮可以将结果复制到剪贴板

注意事项

  • 确保输入的JSON字符串格式正确
  • 字段路径区分大小写
  • 数组索引从0开始
  • 通配符*会匹配所有字段或数组元素
  • 正则表达式需要使用JavaScript语法
  • 如果字段不存在，会提示"未找到指定字段"
  • 本工具在浏览器本地运行，数据不会上传到服务器
