# duck-rush(冲鸭)

一款自用的Shell工具箱，开箱即用

# 使用说明

进入到工程目录，执行 `python install.py`，即可安装完成

# 目录说明

- `install.py` 安装程序，从这里开始，如果编辑了脚本，需要重新安装
- `src` 工具的源码
- `lib` src中的工具依赖的一些类库
- `test` 一些命令的使用示例
- `local` 本地脚本，安装的时候会加入系统PATH，但是不会提交到仓库
